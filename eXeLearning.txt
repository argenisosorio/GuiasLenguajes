=========================
Guía de eXeLearning by dM
=========================

eXeLearning es un programa libre y abierto para crear contenidos educativos de
una manera sencilla.

Instalación
===========

Realizaremos la instalación en GNU/Linux Debian 11.

Nota
====

Usaremos $ para describir los comandos que se usaran con usuario regular.
Usaremos # para describir los comandos que se usaran con superusuario.

1) Descargar el .deb desde: https://exelearning.net/descargas/ Para este
ejemplo hemos descargado es el archivo intef-exe_2.6_all.deb.

2) Ejecutamos la instalación del archivo con:

# dpkg -i intef-exe_2.6_all.deb

====================
Formatos de archivos
====================

eXeLearning permite exportar las lecciones en formato .zip, llamado también
archivo scorm, es el que se sube en la plataforma chamilo, también se puede
exportar en formato .elp, este es el formato para ser leído en eXeLearning.
