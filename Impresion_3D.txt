Software para slicer software for 3d printing
=============================================

El software slicer para la impresión 3D convierte un modelo 3D digital en una
serie de capas (o "rebanadas") que la impresora 3D puede leer y fabricar. Su
función principal es preparar el modelo para que la impresora lo construya capa
por capa, lo que se conoce como fabricación aditiva.


Orca Slicer
===========

Orca Slicer es un programa gratuito de corte para impresión 3D creado por
SoftFever y basado en Bambu Studio. Traduce las instrucciones de impresión a
código G para producir impresiones de alta calidad. Aunque es similar a Bambu
Studio, se diferencia por sus funciones avanzadas, como perfiles de impresora
prediseñados para muchas impresoras 3D populares y herramientas de calibración
automática.


El código G
===========

Es un lenguaje de programación específico que se utiliza para dar instrucciones
a máquinas controladas numéricamente por computadora (CNC), como las impresoras
3D, fresadoras y cortadoras láser. Es el formato de archivo que el software
slicer crea y que la impresora 3D utiliza para fabricar un objeto.
